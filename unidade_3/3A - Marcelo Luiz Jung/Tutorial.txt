Passos para um hello world com o flutter

1º: https://flutter.dev/docs/get-started/install
2º: Faça o download da SKD do flutter, versão 2.0.5
3º: Descompacte o arquivo do paço 2, entre na pasta que corresponde ao arquivo descompatado e faça uma cópia
4º: Cole a pasta copiada no passo 3 no disco C: ou qualquer outro disco desejado, desde que seja na raiz
5º: Acesse as variáveis de ambiente do SO, nas variáveis do sistema procure a variável Path, edite e adicione mais um local
O local a ser adicionado na variável Path deve ser o endereço do passo 5. Exemplo: C:\src\flutter\bin
6º: Para ver se o flutter foi instalado corretamente, execute o comando "flutter doctor", se aparecer algum erro retorne para o 1º passo
7º: Para criar um projeto com o flutter execute o comando "flutter create hello_world"
8º: Abra a pasta do projeto criado no passo 7 usando o VS Code, e instale a extesão "Flutter", é a primeira que aparecer
9º: Para rodar o projeto, após ter feito o passo 8 basta apertar "CTRL + F5", se aparecer algo na barra superior do VS code basta clickar em "CREATE NEW"

Obs.:
- Para depurar é simples, só colocar o breakpoint no VS Code e rodar, pra rodar vs code em mode de depuração é só apertar F5
- Ele abre direto no navegador, é mais simples e da pra usar o modo de compatibilidade

Sucesso e glória!
